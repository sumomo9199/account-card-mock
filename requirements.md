# どんバズくん 競合検索画面 モック要件定義書

## 概要
- **目的**: デザイン調整用のモック作成
- **対象画面**: 競合検索画面（アカウントカード一覧）
- **形式**: 単一HTMLファイル（ブラウザで直接開ける）
- **挙動**: レスポンシブ対応（画面幅に応じてカード並び数が変化）

---

## 1. 共通スタイル

### 1.1 フォント
- **フォントファミリー**: `Lato, "Noto Sans JP", sans-serif`
- **ベーステキストカラー（text-main-black）**: `#333333` / `rgb(51, 51, 51)`

### 1.2 グラデーション
- **btn-primary-gradient / bg-donbuzz-gradient**:
  - `linear-gradient(124deg, #E846C2 0%, #FF5036 100%)`
  - `linear-gradient(124deg, rgb(232, 70, 194) 0%, rgb(255, 80, 54) 100%)`

### 1.3 背景色
- **メインコンテンツエリア**: `#FFFDFE`（非常に薄いピンク/ほぼ白）

### 1.4 ブレークポイント
- **モバイル**: デフォルト
- **デスクトップ（md）**: 768px以上

---

## 2. レイアウト構成

```
+------+-----------------------------------+
|      |         ヘッダー (70px)           |
|      +----------+------------------------+
|サイド|          |                        |
|バー  |フィルター|    カードグリッド        |
|(72px)|パネル    |                        |
|fixed |(約220px) |    (残り幅)            |
|100vh |          |                        |
+------+----------+------------------------+
```

- **ヘッダー＋コンテンツ**: サイドバーの右側（`margin-left: 72px`）に配置

## 10. 実装上の注意点
