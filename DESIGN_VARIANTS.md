# アカウントカード デザインバリエーション

## このページについて

**対象機能**: どんバズくん - 競合検索のアカウントカード

**目的**: カードデザインのバリエーションを比較検討するためのページです。各バージョンを見比べて、どのデザインが良いかフィードバックをください。

**注意点**:
- 見た目（UI）のみのモックです
- フィルターパネルや検索機能は動作しません
- モバイル表示はブラウザの開発者ツールで確認してください

## エンジニア向け: CSS変更ポイント

### レスポンシブ分岐
```css
/* モバイル: デフォルト */
/* デスクトップ: @media (min-width: 768px) */
```

### v7以降のモバイル専用変更
v7〜v9はモバイルのみ変更。デスクトップCSSは変更不要。

### 主要CSSプロパティ
- `grid-template-columns: repeat(auto-fill, minmax(Xpx, Xpx))`
- `-webkit-line-clamp: 2` （名前2行表示）
- `font-size: 0.8em` （単位縮小）
- `margin-bottom: -Xpx` （余白調整）
